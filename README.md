YouTube Clone

A front-end clone of the YouTube homepage built using HTML and CSS. This project replicates the core layout of YouTube, including the header, sidebar, and responsive video grid.

 Features


Header with logo, search bar, voice search, upload, apps, notifications, and user profile icons
Sidebar navigation
Responsive video grid layout
Video cards with thumbnail, duration overlay, channel picture, title, channel name, and view count


 Built With


HTML5
CSS3
Google Fonts (Roboto)
